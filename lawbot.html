Labor LawBot

AI-Powered Labor Law Precedent Assistant

Overview

Labor LawBot is an AI-powered chatbot designed to provide workers and practitioners with precise and reliable information based on labor law precedents. Using a dataset of 1,089 labor law cases spanning from 2010 to 2023, the system offers case-specific insights to streamline workplace conflict resolution and reduce emotional distress.

Impact: Labor LawBot ensures accessibility to legal knowledge for individuals with little or no legal background while aiding professionals by simplifying complex case searches and interpretations.

Process

Labor LawBot was developed through a structured process to ensure accuracy and usability, leveraging state-of-the-art AI and data management tools.

1. Data Collection

Gathered 1,089 labor law cases from 2010 to 2023, focusing on diverse workplace challenges. The data was cleaned, formatted, and verified for reliability. Metadata such as case numbers, rulings, and summaries were added for structured indexing.

2. Indexing & Semantic Search

Utilized Azure AI Search to index labor law cases. A semantic layer was added to improve natural language query handling, allowing users to search using everyday language while retrieving precise case references.

3. GPT Model Integration

Integrated OpenAI GPT to generate detailed and context-aware responses. Adjusted system parameters to ensure the chatbot aligns its answers with indexed cases, maintaining accuracy and trustworthiness.

4. Web Application Development

Developed a user-friendly web application using Gradio, allowing users to interact with the chatbot seamlessly. The interface includes simple input fields for queries and options for detailed follow-up searches.

5. Testing & Optimization

Conducted iterative testing with real-world queries from workers and HR practitioners. Refined the model to handle edge cases and adjusted the data indexing pipeline for consistent results.

6. Deployment

Deployed the chatbot and search features on a scalable Azure environment. Ensured seamless integration with cloud services for efficient data retrieval and future scalability.
